Basics

Auto Chess complete and fully functional game,
it has been created with simplicity in mind.
Great as a starting point to create your own Auto Chess game.
All the scripts are attached to the Script Gameobject In the Hierarchy window.
Basic game rules and Champions can be changed by adjusting public variables.
Core game changes can be done by changing the source code.
Source code uses MVC design and can be easly expanded on.

Champions

Existing Champions and ChampionTypes are located in the Champions and ChampionTypes folder.
To Create new Champion or ChampionType go to Assets Menu -> Create -> Auto Chess -> Champion or ChampionType.
Champions can be customised with this two classes from the editor.
For detailed options check out : Champion and ChampionType documentation.
All Champions and ChampionTypes need to be assigned to the GameData script to be recognised by the ChampionShop.

Packages

Auto chess uses Post Processing package for better visuals.
To install it go to Window Menu -> Package Manager and install Post Processing
